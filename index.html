Roomie

Roommate Finding Mobile Application

A reliable, modern, and user-friendly housing matching platform for young professionals and students living in big cities. Developed with Flutter and Firebase technologies, offering real-time messaging and smart filtering features in a cross-platform mobile application.

Secure & Modern Platform

Roomie features a multi-layered security structure with Firebase Security Rules. Users can only access the data they need, and all operations require authentication.

The intuitive interface designed with Material Design 3 principles prioritizes user experience. With accessibility features compliant with WCAG 2.1 standards, it's a platform accessible to everyone.

Three Main Housing Categories

Roomie offers three main housing categories to suit different needs:

Dormitory Listings: A specialized listing system for student dormitories and boarding houses.

Apartment Listings: Detailed filtering with 40+ feature fields for apartments and flats.

Detached House Listings: Location-based search and matching for independent houses and villas.
