# Roomie Tanıtım Web Sayfası

Bu proje, Roomie mobil uygulaması için basit ve modern bir tanıtım web sayfasıdır.

## Görselleri Ekleme

Görselleri `images` klasörüne aşağıdaki isimlerle ekleyin:

### Zorunlu Görseller:
- **logo.png** (veya .jpg) - Uygulama logosu (header'da gösterilir)
- **screenshot1.png** - Login ekranı görseli
- **screenshot2.png** - Ana sayfa görseli  
- **screenshot3.png** - Premium özellikler görseli
- **screenshot4.png** - Ek tanıtım görseli (isteğe bağlı)
- **qr-code.png** - QR kod görseli (sayfanın en altında)

### Dosya Formatları:
- PNG veya JPG formatında olabilir
- Önerilen boyutlar:
  - Logo: 500x500px (kare)
  - Ekran görüntüleri: Mobil ekran oranında (9:16)
  - QR Kod: 500x500px (kare)

## Kullanım

1. Görselleri `images` klasörüne ekleyin
2. `index.html` dosyasını bir web tarayıcısında açın
3. Sayfa otomatik olarak görselleri yükleyecektir

## Notlar

- Görseller yüklenmezse, placeholder metinler görünecektir
- Sayfa responsive tasarıma sahiptir (mobil uyumlu)
- Tüm görseller otomatik olarak boyutlandırılır
